# 💌 Pedido de Madrinha de Turma

Um site interativo desenvolvido em HTML, CSS e JavaScript para realizar um pedido especial de madrinha (ou padrinho) de turma de forma criativa, emocionante e divertida.

---

## ✨ Demonstração

O site apresenta uma experiência completa:

- 📩 Envelope animado para abertura da carta
- 📝 Carta personalizada
- 💛 Motivos especiais para o convite
- 🎓 Pedido oficial em formato de lousa
- 😂 Botão "Deixa eu pensar..." que foge do cursor
- 🎉 Animação de comemoração com confetes
- 📱 Layout responsivo

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 📂 Estrutura

```
index.html
styles.css
script.js
```

---

## ⚙️ Como usar

1. Faça o download do projeto.

2. Abra o arquivo

```
index.html
```

em qualquer navegador.

Não é necessário instalar nenhuma dependência.

---

## ✏️ Personalização

Basta editar o arquivo `index.html` e alterar:

- Nome da professora
- Nome da turma
- Data
- Escola
- Texto da carta
- Motivos do convite

Todo o código já está comentado para facilitar a edição.
